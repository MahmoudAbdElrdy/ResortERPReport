ComboBox - Overview











IntegralUI ComboBox is a native AngularJS directive which acts as a dropdown list component. Using options in right panel, you can check out some of the built-in properties of the ComboBox directive:

  • enabled - determines whether the ComboBox is enabled or disabled
  • integralHeight - when true, the height of the dropdown list is based on items number shown in whole
  • rtl - changes the ComboBox layout to left-to-right or right-to-left
  • showIcons - determines whether items are displayed with their icon
  • dropDownWidtht - specifies the width of the dropdown list, which can be larger or equal to ComboBox width
  • dropDownHeight - specifies the height of the dropdown list
  • maxDropDownItems - determines how many items are visible in dropdownlist
