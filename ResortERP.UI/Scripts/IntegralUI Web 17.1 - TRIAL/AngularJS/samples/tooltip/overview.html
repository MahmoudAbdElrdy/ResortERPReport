IntegralUI Tooltip is a native AngularJS directive which displays a tooltip. It can be attached to any HTML element or other directive. Using options in right panel, you can check out some of the built-in properties of the Tooltip directive:

  • enabled - determines whether the Tooltip is enabled or disabled
  • showMarker - when true, an arrow marker will be displayed for the Tooltip window
  • title - title of the Tooltip, can contain multiple lines of text or any custom HTML
  • position - determines where the Tooltip will appear: above, below, left or right side of the target element or at mouse position
  • initialDelay - the time in miliseconds prior Tooltip is shown
  • autoPopDelay - the time in miliseconds for Tooltip to remain visible

By default Tooltip is positioned at mouse cursor position. If a title is not provided, the tooltip will not appear.

The appearance of the tooltip is fully customizable. This directive has a separate CSS classes, which can be modified on your side.
