IntegralUI Web for AngularJS

17.1

IntegralUI Web is a suite of advanced AngularJS directives. All components have many features like: fully customizable appearance, optimized for fast data loading and update, improved drag and drop functionality, animations, support for right-to-left layouts, themes and much more.

What's included

  • Accordion - A list of expandable groups with header and content panel
  • CheckBox - Native AngularJS directive that represents a check box
  • ComboBox - Native AngularJS directive that represents a combo box
  • ContextMenu - Adds a custom context menu to other AngularJS Directives or HTML elements
  • Grid - Fast Grid component that displays tabular data sets
  • Menu - Allows you to easily navigate among pages in your web apps
  • SlideBar - Animates slides composed of images or custom content
  • TabStrip - Allows you to create tabbed user interface in your web apps
  • Tooltip - Adds a customizable tooltip to other AngularJS Directives or HTML elements
  • TreeGrid - A Grid component that shows hierarchical data structures in multiple columns
  • TreeView - Fast data loading and update of hierarchical structures

What's New

  • Grid and TreeGrid has built-in context menu functionality
  • Filter data using case insensitive matching criteria
  • Improved performance when grid size changes
  • ----------------------------------------------------
  • Auto-Size option that allows you to use the browser or parent container scrollbar to scroll the TreeView or Grid content
  • Fast vertical scrolling in Grids with large data sets
  • Setting groups in Grid from code
  • Fixed Rows and Items on Top and Bottom Side
  • Improved overall performance
  • Option to show/hide check box for items
  • Speed levels for mouse-wheel: veryfast, fast, normal, slow and veryslow
  • Option to change appearance of fixed rows or items using custom CSS styles
  • Fast hovering over rows when grid has huge size
  • New properties: autoSize, displayMode, groups, mouseWheelSpeed
  • New events: clearGroups, groupAdding, groupAdded, groupChange, groupRemoving, groupRemoved
  • New methods: addGroup, insertGroupAt, insertGroupAfter, insertGroupBefore, clearGroups, removeGroup, removeGroupAt
  • and more ...

What's Fixed:

  • dataFields property now correctly binds custom field names as a replacement for default names for items and rows
  • Fixed the issue when collapsing of the last group caused all groups to expand in the Grid
  • Fixed the problem that causes first selected item to change while holding SHIFT key during multi selection
  • ----------------------------------------------------
  • Fixed the issue that prevents selection of an item or row when clicked twice without holding CTRL or SHIFT key
  • In multi-extended selection mode, now selectedItems and selectedRows return a correct list of selected items and rows, when they are selected without holding CTRL or SHIFT key
  • During filtering row with hasChildren set to true no longer shows expand box, when there are no visible child rows
  • selectionChanged event now fires whenever selected items or rows are changed
  • Other minor bug fixes
